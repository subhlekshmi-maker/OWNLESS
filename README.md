## Problem Statement
Students and individuals often buy items that are rarely used, leading to waste and unnecessary spending. There is no simple platform focused on borrowing instead of owning.

## Solution
OWNLESS is a platform that allows users to **borrow and lend items** within a trusted community, promoting reuse, affordability, and sustainability.

## Features
- User authentication
- List items for lending
- Browse items available for borrowing
- Category-based item search
- Simple and clean UI

## Tech Stack
- HTML
- CSS
- JavaScript
- Supabase (Backend & Authentication)

## How to Run the Project
1. Clone the repository
2. Open the frontend:
   - Open `index.html` in a browser
3. Backend:
   - Supabase is used as a backend service
